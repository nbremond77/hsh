Project goals

First goal: Follow-up the power consumption, the temperature, and other key parameters of my house.

Second goal: Control some appliances in order to improve the energy efficiency, and maybe make life easier.

Architecture

A set of remote sensors and actuators is connected to a central server using a wireless network.

The central server monitors energy, temperature, etc... parameters.

The remote sensors can determine:

The local temperature, relative humidity, pressure... The power consumption of the appliance, or of a part of my electrical grid Some remote actuators can be used to control appliances:

Air conditioning Water pump etc...

Hardware

Remote sensors

Remote sensors - Nucleo STM32F411RE or STM32L152RE based - called "Multisensor board":

  • Radio interface IEEE802.15.4, supporting 6LowPan, for the wireless network
  • 1 x AC Current and Voltage ADC inputs used to calculate AC power
  • 1-Wire interface for DS18B20 temperature sensor or
  • Interface for the DHT22 humidity and temperature sensor
  • BMP180 pressure sensor interface
  • IR receive interface
  • IR transmit interface
  • RS485/DMX interface
  • 2 x digital outputs
  • battery powered with step-up power supply

Remote sensors - Arduino due based - called "Energy monitor board":

  • Radio interface IEEE802.15.4, supporting 6LowPan, for the wireless network
  • 11 x AC Current and Voltage ADC inputs used to calculate AC power
  • 1-Wire interface for DS18B20 temperature sensor or
  • Interface for the DHT22 humidity and temperature sensor
  • BMP180 pressure sensor interface
  • IR receive interface
  • IR transmit interface
  • 2 x RS485/DMX interfaces
  • 8 x digital outputs
  • DC powered

Main server

Raspberry Pi with:

  • Radio interface IEEE802.15.4, supporting 6LowPan, for the wireless network
  • Ethernet connection to the home ethernet network

Software

Remote sensors

Based on a real time operating system supporting IPv6 and 6LowPan wireless network.

Main server

Based on Linux, supporting Ethernet (IPv4 or IPv6) and IPv6 6LowPan network interfaces. Using EmonCMS monitoring server.
